# LaughingMan

> Adds a Laughing Man mask from "Ghost in the Shell: Stand Alone Complex" on faces in a video.

## Installation

1. Requires Python 3.8+
1. Requires a LaughingMan logo with alpha transparency which was downloaded from wikipedia

## Usage

```bash
python3 mask.py
```
